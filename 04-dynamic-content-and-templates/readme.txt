Templating Engines:
- putting dynamic content in our html page, we use templating Engines
- in simple terms, its an html template where data are added to placeholders declared.

- node/express content along with templating engine replaces the placeholders/snippets with html content.
- the result would be dynamically generated html file which is then sent back to your users.

Available Templating Engines:
- EJS  - use normal HTML and plain JS in your templates.
- Pug (Jade)    - use minimal HTML and custom template language
- Handlebars    - use normal HTML and custom template language
